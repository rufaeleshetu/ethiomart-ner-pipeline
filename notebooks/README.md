# Ethiomart NER Pipeline

This project builds an Amharic NER model for extracting product, price, and location information from Telegram-based e-commerce messages in Ethiopia.

## Folder Structure

- `scripts/`: Scrapers and preprocessing scripts
- `data/raw/`: Raw Telegram data
- `data/cleaned/`: Cleaned and tokenized messages
- `images/`: Downloaded product images
- `notebooks/`: Jupyter exploration notebooks

## Getting Started

1. Set up your Telegram API credentials.
2. Run the scraper to collect messages.
3. Preprocess the text and prepare labeled data.
